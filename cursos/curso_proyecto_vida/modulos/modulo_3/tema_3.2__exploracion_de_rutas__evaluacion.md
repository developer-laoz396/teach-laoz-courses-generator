# EVALUACIÓN DEL TEMA 3.2: RUTAS ACADÉMICAS

## SELECCIÓN DE RUTA ADECUADA

*Para cada perfil, recomienda la ruta más lógica (Universidad, Técnica/Tecnológica, Emprendimiento, Fuerzas Armadas).*

1. **Perfil A**: Le encanta la teoría, leer libros de 500 páginas, quiere investigar curas para enfermedades y tiene paciencia para estudiar 10 años. -> (**Universidad / Doctorado**)
2. **Perfil B**: Quiere empezar a ganar dinero rápido, le gusta arreglar motores, se aburre en clases teóricas largas. -> (**Técnica**)
3. **Perfil C**: Odia recibir órdenes, tiene una idea de negocio, es arriesgado y tiene unos ahorros para invertir. -> (**Emprendimiento**)
4. **Perfil D**: Valora el orden, la jerarquía, servir al país y la estabilidad física y laboral total. -> (**Fuerzas Armadas/Policía**)

## VERDADERO O FALSO

1. **(F)** La única forma de tener éxito en la vida es tener un título universitario.
2. **(V)** Un año sabático bien planificado puede ahorrar dinero y frustraciones futuras.
3. **(V)** Las carreras técnicas suelen tener una duración más corta y un enfoque más práctico que las universitarias.
